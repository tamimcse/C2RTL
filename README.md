C2RTL is a high-level synthesis (HLS) tool that generates verilog RTL from C code. It's primarily designed for generating RTL code for FIB lookup algorithms.

Platform

Linux.

Requirements:
============= 
No external library is needed.

* Build:

`make`

* Run:

`make cptrie`
